# CYK Algorithm Simulator

This project implements the CYK (Cocke–Younger–Kasami) algorithm to check whether a given string belongs to a context-free grammar in Chomsky Normal Form (CNF).

## Features

* CYK table construction (triangular form)
* Step-by-step execution
* Automatic acceptance/rejection detection
* Structured parse tree visualization
* Clean and interactive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the repository
2. Open `index.html` in any browser

## Output

* Displays CYK parsing table
* Shows whether string is accepted or rejected
* Generates a parse tree for visualization
